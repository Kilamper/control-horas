# Control de Horas

## Contributing

Please check out the [CONTRIBUTING.md](./CONTRIBUTING.md) document to learn about the contribution process.

## Getting Started

### Clone the repository

```bash
git clone <git url>
```

Enter the project directory

```bash
cd control-de-horas
```

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm run dev
```

## Commands

You can find a cheatsheet with the most common commands used in this project in the file [COMMANDS.md](./COMMANDS.md).

## Project Structure

Inside of your Astro project, you'll see the following folders and files:

```text
/
├── public/
│   └── favicon.svg
│   └── [All the images, fonts, and other static assets]
├── src/
│   ├── components/
│   │   └── [All the components]
│   ├── layouts/
│   │   └── Layout.astro
│   │   └── [All the layouts]
│   └── pages/
│       └── index.astro
│       └── [All the main pages]
├── package.json
└── [Rest of the files]
```
